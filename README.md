# SMC – Garbled Circuits Implementation

This project implements **Yao’s Garbled Circuits**, a foundational protocol in Secure Multi-Party Computation (SMPC). The project is divided into two parts: the construction of a cryptographically secure NAND gate and a practical comparison of oblivious vs. leaky circuit evaluation.

## Files
- `garbled_nand.py` – **Assignment 1**: Implementation of a single garbled NAND gate.
- `garbled_circuits_practice.py` – **Assignment 2**: Comparison of oblivious and leaky array equality checks.

## Requirements
- **Python 3.8+**
- No external packages required (uses standard libraries: `hashlib`, `os`, `secrets`, `time`).


## Installation & Usage

1. **Clone or download** the repository.
2. **Run Assignment 1** to verify the core logic of a single garbled gate:
   
```bash
python garbled_nand.py
